# Competitive_Coding-Algo
The code practice related to testing,Algo and competitive coding

# Running the source code
Code compiles and run automatically with just a keyboard shortcut in vscode.
The code is wrapped first inside a timer library that calculates the execution time.
Keyboard shortcut `ctrl + shift + b`

# Copiling the TaskRunner
```bash
cd path-to-TaskRunner.cpp ; g++ -std=c++17 TaskRunner.cpp -o TaskRunner.exe
```
