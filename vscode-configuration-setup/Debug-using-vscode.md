# VScode Debugger
- Simple applications doesn't require separate configuration
- Create `launch.json` file for complex debug scenarios. Get more information about creating a [debug configuration](https://code.visualstudio.com/docs/debugtest/debugging-configuration). 
- >[!Tip]
  [Use copilot for generating debug configuration](https://code.visualstudio.com/docs/debugtest/debugging-configuration#_generate-a-launch-configuration-with-ai)
- Default debugger support: `.js, .tsx and Node.js`
