## ğŸƒ Running the Code

This repository uses a custom TaskRunner system for compilation and execution. All solutions can be run using VS Code with a simple keyboard shortcut.

### Quick Start
1. Open any solution file in VS Code
2. Update `inputf.in` in the repository root with your test input
3. Press `Ctrl + Shift + B` to compile and run
4. Check `outputf.in` for the output

### Compilation Details
- **C++**: Compiled with `-std=c++17 -O2` flags
- **Java**: Compiled with memory-constrained JVM settings
- **Python**: Runs with `python3` interpreter
- **JavaScript**: Runs with Node.js

For more details, see the [main repository README](../../README.md).
